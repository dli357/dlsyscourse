# Homework 0
Public repository and stub/testing code for Homework 0 of 10-714.

## Setup (Windows)

1. Install Build Tools for Visual Studio (see https://visualstudio.microsoft.com/downloads/).
1. Install Python3.
1. Install `pybind11` by running `pip install pybind11`.
1. [*Optional*] Install `venv` for simplified dependency management by running `pip install virtualenv`.
1. Build `simple_ml_ext.cpp` PyBind library by running `.\make.bat`.
